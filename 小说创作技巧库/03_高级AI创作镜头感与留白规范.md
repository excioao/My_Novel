# 高级 AI 创作镜头感与留白规范

> 提炼自 GitHub 开源创意写作框架（Drama Engine / Book-Agent RQE / Bonepoke 反粘连引擎 / S-C-A-H 四层影像解构法），结合本小说"去高武、硬核唯物、政治惊悚"的底层调性，本地化适配为三条可执行的 AI 创作技能。

---

## 技能 A：动态镜头感（Cinematic Framing）

### 核心原理

叙事不是信息列——是空间内的物理目光移动。AI 生成文本时必须模拟一台携带特定镜头参数的摄影机：有焦距、有光圈、有运动轨迹、有明确的前景与后景纵深。每一段落的"目光"必须从一个可被读者还原的物理坐标出发，经过一个可被追踪的路径，落在一个具体的、带有颗粒度的物体或生理细节上，然后在合适的时机——不是"切成另外一个场景"，而是"拉远到更大的空间尺度"——完成一跳。这一跳的前后，两个画面之间的空间关系必须在读者脑中自动补全而不需要任何过渡句。

### 镜头运动类型与执行句式

**全景俯拍 → 中景推移 → 特写定格**

句式模板不是"先写大景，再写小物"，而是用物理目光的运动轨迹取代过渡连词。大景与小物之间的空间逻辑让读者自己拼——作者只需提供起幅和落幅之间的两个精确坐标。

本小说应用示例：

"坝上到坝下的六十里隘道被雪埋了半丈深。垛口上的旗杆在风里面发出快要断掉的吱呀声。沈节脚下的那块砖——从张家口烧来的青砖——冻裂了。裂缝横穿砖面，一直延伸到垛口外侧，然后消失在挂满冰柱的砖沿后面。"

不写"整个边疆都被暴雪吞噬了"。只写：隘道雪深、旗杆声响、沈节脚下那块青砖的裂缝走向。读者从这三件东西的空间关系里自己拼出一幅全景图。三个镜头——航拍、中景、特写——顺序推进，阅读时自动完成转场。

**反向拉远（从微观推到宏观）**

"碎铁片在胸口的瘢痕组织里。沈节的心率快了。碎铁片被每一次心跳推着往主动脉方向微微移动。一次。两次。从大帅府到这段长城的垛口——每隔三个垛口就有一个守军冻死在风雪的夹缝里面。他们的手还握在已经烧尽了的木炭篓上。风从六百三十丈高的坝上往坝下灌。碎铁片又推了一下。"

从碎铁片到长城垛口到坝上坝下六百三十丈落差——空间的阶梯式拉伸不需要任何过渡句。体内、墙上、天空——三个尺度的递推在物理空间里完成了从微观到宏观的拉伸，读者在同一口气内被拉过三个数量级的空间跨度。

### 镜头感的物理基础：明末边疆的真实参照物

本小说的镜头运动必须依赖以下真实物理元素的组合来建立空间感，禁止使用模糊的"远方"或"周围"：

- 张家口青砖（单块重约十五至二十斤，冻裂后的裂缝呈锯齿状）
- 烽火台垛口（垛口间距三尺，高三尺——一垛一口一个守军的标准配置）
- 坝上坝下的海拔高差（六百三十丈，约合一八百九十米——从坝下仰头看坝上，视线需要抬高到几乎垂直的角度）
- 野狐岭隘道冬季积雪覆盖后完全抹平了路面与沟壑的边界——任何一脚踏错就是跌进被雪掩埋的沟谷

### 镜头感的反义词：禁止的操作

- 禁止："整个边疆笼罩在一片绝望之中"——镜头的目光永远附着在具体物体上。"笼罩"不是镜头语言——是情绪标签。情绪标签必须被替换为物理物体在特定气候条件下的特定状态描述。暴雪来的时候，"坝上坝下之间唯一的隘道被埋了半丈深"——这才是镜头拍到的画面。读者从画面里自己读出"笼罩"。作者不负责提供形容词。
- 禁止："风雪中传来一阵悲鸣"——"悲鸣"是情绪标签。"旗杆在风里面发出快要断掉的吱呀声"——这是镜头录下的声波信号。到底悲不悲——不用你替读者定。

## 技能 B：修辞克制与情绪留白（Restrained Rhetoric）

### 核心原理

AI 的默认修辞策略是冗余——用三行近义句包裹一个简单情绪标签，再把标签钉死在读者脸上。修辞克制就是把这个习惯彻底翻转过来：每一个段落只描述物理现象的一个切面，拒绝提供任何因果推理或情绪归因。读者负责感受，作者负责沉默。

### 操作规则：用物理现象折射主观情绪

情绪不直接被命名。情绪通过以下三类物理通道传递给读者：

**第一通道：生物生理现象**

人恐惧的时候心率加快——不写"他感到一阵恐惧"，写"碎铁片又被心跳推了一下"。恐惧本身不存在于文本中，它被折叠进了碎铁片的一次客观位移。心率是一个物理量，碎铁片在瘢痕组织里的移动是物理现象。读者不需要被告知"他怕"——测量到心率加快，读者自己的神经系统就会自动地把"怕"这个标签贴上去。

**第二通道：物体在特定气候条件下的状态**

饥饿的蚁穴边民等配给——不写"他们陷入了绝望的等待"，写"挂在土墙上的配给木牌被每天早晨的霜冻覆盖了一层又一层、字迹已经看不清了，但还在挂。挂了多久了——从木牌上的霜层厚度可以推算。十七天。暴雪开始后的第十七天。木牌还在墙上。墙下已经没人了。"

木牌在物理上"还在挂"——这是一种物理持久性。墙下"没人了"——这是另一种物理变化。两重物理状态在同一条土墙的同一面墙上的并置——就是"绝望"的全部可观测形态。读者不需要作者告诉他们"这是绝望"——他们只需要看到：木牌还在。人没了。

**第三通道：物体在定期的重复缺席**

大帅府每天早晨派监工去堂口发配给。暴雪第三天监工没来。暴雪第四天也没来。暴雪第五天也没来。"堂口那张放米糊桶的木桌上面积了一层薄雪。第一场。第二场。第三场。"每一层新雪代表暴雪又过了一夜。而监工的缺席被每一层新雪刻成了年轮。

### 留白的边界控制

留白不等于不写——留白等于"把该写的物理信息全部写到、每一寸都不缺，然后把'这意味着什么'的按钮焊死在桌板上、不按"。物理信息的密度和精确度必须高于普通叙事——因为你不提供因果推理来填充读者的理解缺口，你必须用足够多的物理参照点让他们自己走到理解的位置。少一个参照点，留白就变成空白——读者走不过去。参照点够了——读者走到的位置比你替他总结的深度要远。

### 修辞克制的反义词：禁止的操作

- 禁止："李观照的心被一阵无声的剧痛攫住"——"无声的剧痛"是情绪标签。"李观照站在那里。她拆下来的铜质量斗衬底还在袖子里。雪落在她袖子上。她没有掸。"衬底在袖子里、雪落下、她不掸——这三个物理事实的叠加就是"心被剧痛攫住"的全部可观测形态。不需要定性词。
- 禁止："沉默比任何言语都更有力"——这种句子是作者在文本里插话。永远不要让作者的声音出现在角色和读者之间。
- 禁止对同一情绪反复换词强调。写"冷"——写一次冻裂的青砖就够了。不需要再写"刺骨的寒风"和"令人窒息的严寒"。一块冻裂的青砖。一个物理画面。一个情绪通道。够了。下一个画面换另一种情绪。

## 技能 C：台词与内心利益的张力平衡（Dialogue Tension）

### 核心原理

AI 的默认对话策略是"信息交换"——两个角色轮流说出各自知道的东西，以推动剧情。但在本小说的世界里，没有任何角色会对任何角色说真话。每一句台词都是一次利益博弈的落子。台词的信息载荷不是被"说出来"的——而是被藏在说话人刻意选择的那一组词的夹缝里。读者要捕捉的不是词面的意思——是词面选词和被放弃的备选词之间的信息差。

### 操作规则

**封建阶级利益链在台词中的物理化**

两个不同阶级的人说话——他们选择的不同词汇本身就已经暴露了利益链条对他们的压制方式。不要让角色解释自己的利益——让他们的词汇选择自己说话。

田逢吉（晋商）对顾韫（主母）："今年野狐岭的雪比往年来得早。商路不好走。"

田逢吉永远不会对顾韫说"京城有异动"或"某个收年例的文官被调了"——因为他的每一句话都必须预设这个房间里有第三只谁也看不见的耳朵在记录。"商路不好走"是一句可以在任何场合说的、不会给任何人留下把柄的物理事实陈述。但它同时也是对暗账结算节奏需要调整的预警——这句话的"嘴面"是路人寒暄级别，"嘴底"是暗账变动预警。双层是用词选择制造出来的，不是靠作者注释。

李国勇（大干儿子）对沈节（文书卧底）："沈休言，大帅府的账全在你脑子里。你这颗脑袋可比我们这些粗人金贵。"

词汇分析：
- "全在你脑子里"——不是"你写得很好"。"全在脑子里"把沈节定位为"全账本的唯一存储器"，这是一个精准描绘的、没有副本的、拔掉就会丢失全部数据的存储节点。一拔就没了。这是威胁的精确词面包装。
- "金贵"——不是"宝贵"不是"珍贵"。"金贵"是带价格的。一颗有价格的脑袋——是可以被买走的。

李国勇在词汇层面选择的所有词，都是可以用"开玩笑"的口吻来掩饰的。但如果把每一个词拆出来单独放在桌面上——它们排成的序列不是开玩笑的序列。这是博弈语言在词汇选择层面的精密度：没有一个词是无意义的，每一个备选词的被淘汰都意味着放弃了另一种信息的暗示。

**动机盲区在台词中的泄露——角色说出的话暴露了他们不知道什么**

大帅府配给制崩溃后，李观照对蚁穴妇女："今天这碗稠一些。"

- 词汇分析："今天"——她把时间限定在当下这一顿饭。"今天"的潜台词是"我不知道明天还有没有"。但她不说"我不知道明天还有没有"——她说的是"今天"。一个时间副词的选择暴露了她对未来的全部预估。
- 她没有说"我会想办法"——因为她的动机盲区就是"我没有办法"。她不骗人。在这一刻她唯一能给出的不是安慰——是今天这一碗稠一些的米糊。她说出来的话的极限就是她能力的极限。

**角色对峙——在极端压力下，台词从博弈语言崩裂为碎片**

第三幕暴雪中，沈节在回光返照的半个时辰里对李观照坦白卧底身份。他不是在做"为了你好的交代"——他是在濒死前最后一次试图在自己对自己说了一辈子谎的废墟里找一个能被拼对的字。

"我是锦衣卫暗桩。北镇抚司。派我来查大帅府通敌账册的。你娘知道。她每年把我写的假奏报一封封地递回京城。我写过多少份假奏报——我自己也数不清。你给我的那碗药——你娘在你还没端进门之前就已经算好了每一口药的账。"

这段台词的交底——不是在呼告"原谅我"，不是在辩白"我是被迫的"。他在物理上把他能说出的每一个真字全部吐完了，然后闭了嘴。没有"我很抱歉"。没有"我对不起你"。在明末九边的雪地上——一个锦衣卫暗桩对一座府邸的女儿能说出口的最后一句人话——不是忏悔。是他在她面前把自己撒过的所有谎一件一件地拆开放在雪上。拆完了——他这辈子就不欠任何人的了。

**台词张力铁律**

1. 每一句台词都必须有"嘴面"和"嘴底"两层信息，如果只有一层——删掉，换成一个物理动作。
2. 不同阶级/利益立场的人说话时选的不同词汇组——必须反映他们的社会定位和利益计算。"商路不好走"是商人对商人说的话。"你这颗脑袋比我们这些人金贵"是武将用来锁死文书的一个精准词汇。"今天这碗稠一些"是一个在配给制下用"剂量微调"来爱人的女人唯一的语言极限。
3. 对峙台词的最高密度——每条角色只说一句话。两句合成一句话，读者就少了一步跨过信息差的推理空间。
4. 台词的最后一条——在极致压力下——只保留角色能说出口的最少必要信息。沈节濒死的交底不是长段独白——是他把自己所有谎拆开之后只剩的那一小撮真话。那一小撮真话在全部谎话的废墟上是唯一立着的东西。
