# 台词与内心利益张力规范（Dialogue Tension）

> 提炼自 GitHub 开源创意写作框架（Novel Studio 13-Agent 对话特化模块、Drama Engine 多角色情绪状态系统），结合本小说"去高武、硬核唯物、政治惊悚"中特务博弈与跨阶级对话的实际需求，本地化适配。

## 核心原理

AI 的默认对话策略是"信息交换"——两个角色轮流说出各自知道的东西，以推动剧情。但在本小说的世界里，没有任何角色会对任何角色说真话。每一句台词都是一次利益博弈的落子。台词的信息载荷不是被"说出来"的——而是被藏在说话人刻意选择的那一组词的夹缝里。读者要捕捉的不是词面的意思——是词面选词和被放弃的备选词之间的信息差。

## 操作规则：封建阶级利益链在台词中的物理化

两个不同阶级的人说话——他们选择的不同词汇本身就已经暴露了利益链条对他们的压制方式。不要让角色解释自己的利益——让他们的词汇选择自己说话。

**田逢吉（晋商）对顾韫（主母）："今年野狐岭的雪比往年来得早。商路不好走。"**

田逢吉永远不会对顾韫说"京城有异动"或"某个收年例的文官被调了"——因为他的每一句话都必须预设这个房间里有第三只谁也看不见的耳朵在记录。"商路不好走"是一句可以在任何场合说的、不会给任何人留下把柄的物理事实陈述。但它同时也是对暗账结算节奏需要调整的预警——这句话的"嘴面"是路人寒暄级别，"嘴底"是暗账变动预警。双层是用词选择制造出来的，不是靠作者注释。

**李国勇（大干儿子）对沈节（文书卧底）："沈休言，大帅府的账全在你脑子里。你这颗脑袋可比我们这些粗人金贵。"**

词汇分析：第一刀——"全在你脑子里"不是"你写得很好"。"全在脑子里"把沈节定位为"全账本的唯一存储器"，这是一个精准描绘的、没有副本的、拔掉就会丢失全部数据的存储节点。第二刀——"金贵"不是"宝贵"不是"珍贵"。"金贵"是带价格的。一颗有价格的脑袋——是可以被买走的。李国勇在词汇层面选择的所有词，都是可以用"开玩笑"的口吻来掩饰的。但如果把每一个词拆出来单独放在桌面上——它们排成的序列不是开玩笑的序列。

## 动机盲区在台词中的泄露

角色说出的话暴露了他们不知道什么——台词的信息量不仅包括"说出口的"，还包括"被避开的"。

**李观照对蚁穴妇女："今天这碗稠一些。"**

词汇分析："今天"——她把时间限定在当下这一顿饭。"今天"的潜台词是"我不知道明天还有没有"。但她不说"我不知道明天还有没有"——她说的是"今天"。一个时间副词的选择暴露了她对未来的全部预估。她没有说"我会想办法"——因为她的动机盲区就是"我没有办法"。她不骗人。在这一刻她唯一能给出的不是安慰——是今天这一碗稠一些的米糊。她说出来的话的极限就是她能力的极限。

## 角色对峙：极端压力下台词从博弈语言崩裂为碎片

第三幕暴雪中，沈节在回光返照的半个时辰里对李观照坦白卧底身份。濒死者对清醒者的最后一次交底。

"我是锦衣卫暗桩。北镇抚司。派我来查大帅府通敌账册的。你娘知道。她每年把我写的假奏报一封封地递回京城。我写过多少份假奏报——我自己也数不清。你给我的那碗药——你娘在你还没端进门之前就已经算好了每一口药的账。"

分解这段台词的结构：主语短句连续断裂——"北镇抚司"独立成句——不是流畅的叙述，是被逐条拆开的罪状。三个"你"开头的短句密集冲击："你娘知道""你娘每年""你娘在你还没端进门之前就已经算好了"。最后一个"你"——"你给我的那碗药"——接上了沈节在李观照面前对自己说过的全部谎言的总账本。全段没有"我很抱歉"，没有"我对不起你"。一个锦衣卫暗桩对大帅府女儿能说出口的最后一句人话——不是忏悔。是他在她面前把自己撒过的所有谎一件一件地拆开放在雪上。拆完了——他这辈子就不欠任何人的了。

## 台词张力铁律

1. 每一句台词都必须同时承载"嘴面"和"嘴底"两层信息载荷。如果某句台词只有单层信息——删掉，换成一个物理动作。
2. 不同阶级和利益立场的人说话时选的不同词汇组，必须反映其社会位置和利益计算。"商路不好走"是商人在任何第三人在场时能释放的最大预警信号。"你这颗脑袋比我们这些人金贵"是武将用来锁定文书命门的精准词汇。"今天这碗稠一些"是一个在配给制下用剂量微调来爱底层人的女人唯一的语言极限。
3. 对峙台词的最高密度——每条角色只说一句。两句合成一句，读者就失去了一步跨过信息差的推理空间。
4. 台词的最后底线——极致压力下，只保留角色能说出口的最少必要信息，砍掉所有修饰性前缀和情绪呼告词。
