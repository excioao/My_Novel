## 六、流水线（run_workflow.py）

- 指挥审计：DeepSeek V4 Pro（`deepseek-v4-pro`，`https://api.deepseek.com/v1`）
- 文学创作：Kimi K2.5（`kimi-k2.5`，`https://api.moonshot.cn/v1`，temperature 固定 1.0）
- 流程：DeepSeek 生成任务卡 → Kimi 撰写初稿 → Python 审计五维扫描 → 致命违规直接驳回 / 警告累计 3 条驳回 → 最多 3 次驳回循环 → 审计通过后写入 `草稿箱/第XX章_标题.md` → 每章自动 git commit + push
- 运行方式：`python run_workflow.py`。需在 `main()` 中设置场景列表
- 进度显示：streaming 模式，每 200 字刷新字数，驳回时显示第几次重试
